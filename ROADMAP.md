# 开发路线图

> 最后更新: v0.1.000（测试/Beta）

## 版本规划

### ✅ v0.1.000（测试/Beta）— 已完成

**核心玩法闭环**

- [x] 坦克移动系统（尼基金转向 + 功率分配）
- [x] 炮塔控制（TPS/AIM 双模式 + 自由视角）
- [x] 开火与装填（弹药切换 + 测距 + 弹道计算）
- [x] 火控 HUD（自定义布局 + FOV 缩放）
- [x] 碰撞与伤害（装甲区域 + 穿透/跳弹）
- [x] 对象池（炮弹池化回收）
- [x] 音频系统（FMOD 引擎状态机）
- [x] 悬挂系统（悬挂臂 + 轮子旋转）
- [x] 天气系统
- [x] UI 系统（暂停/设置/HUD/瞄准镜）

---

### 🔜 v0.2.000（测试/Beta）— 计划中

**任务与内容扩展**

优先级从高到低：

- [ ] **任务系统**
  - 任务目标定义（ScriptableObject）
  - 进度追踪与条件判定
  - 任务完成/失败反馈
  - 任务链支持

- [ ] **多坦克支持**
  - 坦克选择界面
  - 不同坦克数据配置
  - 载具切换逻辑

- [ ] **伤害视觉反馈**
  - 车体损伤模型（动态破损）
  - 起火/爆炸粒子特效
  - 命中方向指示器

- [ ] **AI 敌方坦克**
  - 基础 AI 行为树（巡逻/索敌/追踪/开火）
  - 难度分级
  - 多 AI 协同

- [ ] **地图系统**
  - 小地图（MiniMap）
  - 战术标记
  - 地图缩放与平移

- [ ] **设置持久化**
  - 图形设置保存/加载
  - 音频设置保存/加载
  - 控制绑定保存/加载
